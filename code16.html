  • Use of (in) operator in table

    • IN is an operator in SQL, which is generally used with a WHERE clause. Using the IN operator, multiple values can be specified.

    It allows us to easily test if an expression matches any value in a list of values.

    IN operator is used to replace many OR conditions.


Syntax:

select * from table_name where column_name in (data1,data2,..);
