  • How to Insert data in Views


Actually data is not inserted in the view. View is created with the help of table.The of view is shown as the data stored in table.In other words we can say view is not a real entity. It is a virtual entity.

The syntax of insert data in view as following:

Syntax :

insert into view_name values(val1,val2,...);

Output:

Inserted data shown in view

Inserted data shown in table as in view
